# Image-Card Deck Route

Use this route for social cards, cute/NPC-style visual decks, image-first summaries, shareable explainers, or fast first-pass synthesis from messy source material.

## Production Principles

- Treat each slide as a finished poster/card.
- Use fixed ratios: 16:9 for presentations, 4:5 for social feeds, 1:1 for square cards, 9:16 for stories.
- Text can be rasterized only because image output is intentional. If the user asks for editable PPT, use the editable PPTX route.
- Use fewer words than a PPTX report. Put detail in captions, notes, or a companion document.
- Keep a consistent card system: title position, character/illustration rules, source line, numbering, and safe margins.

## Workflow

1. Extract the source into 5-12 key cards: hook, problem, facts, contrast, process, example, takeaway, action.
2. Choose tone: cute/NPC, clean notebook, premium editorial, educational explainer, or meme-adjacent.
3. Write each card with one dominant sentence and one supporting detail.
4. Generate or source visuals only when they carry meaning; avoid generic filler.
5. Export cards as PNG/JPEG, then optionally wrap them into PPTX or PDF for sharing.
6. Render a contact sheet for quick review when there are many cards.

## Card Design Standards

- Keep text inside safe margins; test the longest Chinese and English lines.
- Use high-contrast text over images; add a solid overlay if needed.
- Avoid tiny footnotes unless the card is meant for zoomed reading.
- Use consistent numbering and section markers.
- Preserve brand marks and legal/source lines when required.

## Source Handling

- Use user-provided files, licensed assets, or public sources.
- Do not bypass access restrictions.
- Summarize and cite rather than copying long passages.
- For research cards, keep a source ledger so claims can be traced.

## Final Checks

- View cards at the size they will be shared.
- Confirm no text is clipped after export.
- Confirm all generated images match the intended subject and do not contain broken text.
- Confirm the wrapper PPTX/PDF, if created, preserves image quality.
