# Style Systems

Pick one style family early. Do not mix families unless the deck deliberately shifts tone by section.

## Executive Editable

For board, sales, strategy, investor, and report decks.

- Quiet grid, strong sectioning, concise action titles.
- Editable charts/tables, restrained color, high information clarity.
- Best output: editable PPTX.

## Editorial HTML

For visually ambitious talks, launches, and public presentations.

- Large typographic moments, full-bleed images, confident asymmetry.
- Browser-native layout and transitions.
- Best output: HTML stage deck or HTML-first design later converted to PPTX.

## Swiss Grid

For crisp offline talks, education, thought leadership, and design-forward sharing.

- Strong grid, disciplined palette, numeric markers, whitespace.
- Keyboard-first delivery and slide-sorter clarity.
- Best output: HTML stage deck, or PPTX if editability is required.

## Practical Presenter

For rehearsed talks, workshops, course material, and timed speeches.

- Notes, transcript, timer, progress, section durations, and speaker prompts.
- Design can be simple; delivery support is the differentiator.
- Best output: HTML stage deck or PPTX with detailed notes.

## Cute Visual Cards

For playful explainers, social sharing, consumer education, and lightweight summaries.

- Character-like tone, friendly visual metaphors, short copy, poster-card rhythm.
- Best output: image-card deck or PPTX/PDF wrapper.

## Notebook Synthesis

For turning many sources into a first readable structure.

- Cards or slides emphasize extraction, clustering, and first-order clarity over final polish.
- Best output: image-card deck, outline, or draft PPTX ready for redesign.

## Style Selection Heuristics

- If the user says "可编辑", "PPTX", "客户能改", choose Executive Editable.
- If the user says "审美", "网页", "HTML", "很炸", choose Editorial HTML or Swiss Grid.
- If the user says "线下分享", "快捷键", "演讲", choose Swiss Grid or Practical Presenter.
- If the user says "计时器", "逐字稿", "讲稿", choose Practical Presenter.
- If the user says "小红书", "图片", "卡片", "可爱", choose Cute Visual Cards.
- If the user provides messy articles, PDFs, links, or notes and asks for a first version, choose Notebook Synthesis first, then upgrade to another style.

## Palette And Typography

- Use one dominant neutral, one accent, and one support color; add data colors only when charts need categories.
- Avoid a deck dominated by one hue family unless it is a brand requirement.
- Use two type scales: display titles and content text. Do not scale font size with viewport width in HTML.
- Keep Chinese typography generous: more line height, fewer words per line, and careful punctuation spacing.

## Slide Rhythm

- Alternate between argument, proof, relief, and decision slides.
- Use section dividers to reset attention in long decks.
- Put the densest slide after the audience understands why it matters.
- End with a concrete action, not only a summary.
