# 女娲术语速查表

> 第一次使用女娲时，概念密度可能有点高。这张表把女娲术语和通用术语一一对应，方便快速上手。

## 核心隐喻

| 女娲术语 | 通用术语 | 一句话解释 |
|---|---|---|
| 生灵 | Skill | 被塑造的对象，一件准备公开的作品 |
| 泥 / 土 | 原始素材 | 一个能跑但还没打磨好的 Skill |
| 造物工坊 | 打磨流程 | 女娲的七步方法论 |

## 七步工序

| 女娲工序 | 通用说法 | 核心动作 | 产出物 |
|---|---|---|---|
| **择土** | 前提挑战 / 价值验证 | 先判断这个 Skill 值不值得打磨 | 择土结论（好泥/可用/朽土） |
| **观世** | 竞品调研 / 生态扫描 | 找同类 Skill，分析它们凭什么立足 | 同族对标表（带URL） |
| **定命** | 战略定位 | 纵向看来路去向，横向看行情，交叉定位 | 一句话新定位 |
| **量骨** | 质量审计 / 评分 | 结构+实测+活体三把尺打分 | 九维评分表 + 差距清单 |
| **细塑** | 候选改写 / 优化 | 冻结原版，只改一个面，过验证门才保留 | 可直接替换的改写片段 |
| **绘灵** | README + Showcase 升级 | 把 Skill 包装成别人愿意看、想装的公共资产 | 显灵书 + demo |
| **轮回** | 持续迭代 / 回炉 | 留对标观察清单，下一轮从真实反馈进 | 迭代纪律 + 下一轮入口 |

## 关键机制

| 女娲术语 | 通用术语 | 含义 |
|---|---|---|
| **定坯** | 冻结基线 | 改动前封存原版，所有候选改动都和基线对比 |
| **灵关** | 验证门 | 候选改写必须满足的条件（测试优于基线、首屏10秒说清价值等） |
| **回泥** | 回滚 / 回刀 | 改动没过验证门，回到基线重新来 |
| **灵契** | 出师证书 / 结果卡 | 打磨报告末尾的可截图传播卡片 |
| **显灵书** | README | Skill 的对外展示文档 |
| **接引** | 安装 / 采用 | 用户把 Skill 装到自己的 Agent 上 |
| **同族** | 同行 / 同类 | 解决相同或相邻问题的其他 Skill |
| **命格** | 定位 / 生态位 | 这个 Skill 在市场中该扮演的角色 |
| **暗疾** | 静默失败 | 看起来正常但实际已经失效的问题（如CI全绿但数据停更） |
